# Keyboards

Custom keyboards and keyboard layouts

## Custom Keyboards

### [T-Rexth 36](https://github.com/erhickey/t_rexth_36)

- 36 keys
- Split keyboard
- Ortholinear
- Wireless
- MX switch compatible
- Minimal spacing
- Hot swap sockets
- Tenting puck compatible
